macOS Focus Utility

시야 잡음을 줄이고
지금 필요한 창만 남기세요.

AntiADHD는 macOS 메뉴바에서 동작하는 Focus Mask 앱입니다. 화면의 나머지 영역을 마스킹해 현재 작업 창에 집중할 수 있도록 도와줍니다.

macOS 다운로드 GitHub Releases 기본 링크 릴리스 보기

AntiADHD란?

전역 단축키로 오버레이를 제어하는 경량 메뉴바 앱이며, 창 단위 집중 모드를 지원합니다.

  • macOS 13+ 지원
  • Dock 아이콘 없이 액세서리 앱으로 동작
  • 권한 미허용 시 안전하게 폴백

모드

집중 방식에 맞춰 선택하세요

Current Display

마우스가 있는 모니터만 보이게 하고, 나머지 디스플레이는 마스킹합니다.

Focused Window

현재 포커스 창만 강조합니다. 비선택 영역은 Freeze, Black Overlay, Blur 방식으로 처리할 수 있습니다.

Locked Window

선택한 창을 고정 대상으로 유지하여 커서 이동과 관계없이 계속 보이게 합니다.

단축키

작업 흐름을 끊지 않는 제어

⌥⌘B

집중 마스크 On/Off 전환

⌥⌘L

현재 포커스 창 잠금

⌥⌘⎋

긴급 해제로 오버레이 즉시 해제

권한

창 모드 사용 시 Accessibility 권한 필요

Focused Window와 Locked Window 모드는 손쉬운 사용(Accessibility) 권한이 필요합니다. 권한이 없으면 앱은 자동으로 Current Display 모드로 전환됩니다.
