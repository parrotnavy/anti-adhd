macOS 집중 도구

산만한 화면은 덜어내고
지금 필요한 창에만 집중하세요.

AntiADHD는 macOS 메뉴바에서 실행되는 Focus Mask 앱입니다. 주변 화면을 가려 지금 작업 중인 창에 더 집중할 수 있게 도와줍니다.

macOS 앱 배포 준비가 끝날 때까지 공개 다운로드는 coming soon 상태로 표시됩니다.

AntiADHD란?

전역 단축키로 오버레이를 제어하는 가벼운 메뉴바 앱으로, 창 단위 집중 모드를 지원합니다.

  • macOS 13 이상 지원
  • Dock 아이콘 없이 메뉴바에서 실행
  • 권한이 없으면 Current Display 모드로 안전하게 전환

모드

원하는 방식으로 집중하세요

Current Display

커서가 있는 모니터만 선명하게 남기고, 다른 디스플레이는 가립니다.

멈춤 어둡게 흐리게

Focused Window

현재 창은 선명하게 두고, 주변 영역은 정지 화면처럼 유지하거나 어둡게 또는 흐리게 처리할 수 있습니다.

Locked Window

하나의 창을 고정해 두면, 커서를 다른 창으로 옮겨도 그 창은 계속 선명하게 유지됩니다.

권한

창 모드를 사용하려면 Accessibility 권한이 필요합니다

Focused Window와 Locked Window를 사용하려면 손쉬운 사용(Accessibility) 권한이 필요합니다. 권한이 없으면 AntiADHD는 자동으로 Current Display 모드로 전환됩니다.

FAQ

사용자와 크롤러가 자주 묻는 질문

핵심 정보를 짧고 직접적인 문장으로 정리해 검색엔진, AI 도구, 실제 방문자 모두가 쉽게 이해할 수 있게 했습니다.

AntiADHD는 어떤 앱인가요?

AntiADHD는 지금 집중할 창이나 디스플레이만 선명하게 남기고 주변 화면을 가려 주는 macOS 메뉴바 Focus Mask 앱입니다.

누구를 위한 앱인가요?

글쓰기, 코딩, 문서 읽기, 디자인처럼 화면 잡음을 줄였을 때 더 몰입하기 쉬운 작업을 하는 Mac 사용자를 위한 앱입니다.

어떤 macOS 버전을 지원하나요?

AntiADHD는 macOS 13 이상을 기준으로 설계되었습니다.

어떤 권한이 필요한가요?

Focused Window와 Locked Window에는 Accessibility 권한이 필요합니다. 권한이 없으면 앱은 Current Display 모드로 안전하게 전환됩니다.

지금 바로 다운로드할 수 있나요?

아직은 아닙니다. 현재 웹사이트에서는 macOS 앱 공개 배포 준비가 끝날 때까지 coming soon 상태로 안내하고 있습니다.
